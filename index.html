quick start

get it running

both themes follow the same install pattern. one line to switch between them at any time.

parchment.nvim lazy.nvim
{
  "saeeedhany/parchment.nvim",
  priority = 1000,
  config = function()
    require("parchment").setup({})
    vim.cmd("colorscheme parchment")
    -- or: parchment-ember
    -- or: parchment-manuscript
  end,
}
overcast.nvim lazy.nvim
{
  "saeeedhany/overcast.nvim",
  priority = 1000,
  config = function()
    require("overcast").setup({})
    vim.cmd("colorscheme overcast")
    -- or: overcast-dawn
  end,
}
new machine

one line. full environment.

fresh machine to complete setup. clones dotfiles, builds suckless tools from source, configures neovim, bash, x11, fonts, and scripts. supports arch and minimal distros.

run this on the new machine
$ bash <(curl -fsSL https://saeeedhany.github.io/dotfiles/install.sh)
what gets set up
dotfiles
clones github.com/saeeedhany/dotfiles → ~/.dotfiles
neovim
latest build + symlinks ~/.config/nvim. lazy.nvim auto-installs plugins on first launch.
suckless suite
dwm · st · dmenu · slstatus — all compiled from source with your patched config.h
bash
symlinks .bashrc and .bash_profile — prompt, aliases, utility scripts
fonts & x11
JetBrainsMono Nerd Font, .xinitrc, .Xresources merged via xrdb
scripts + misc
dotfiles/bin → ~/bin  ·  feh · picom · dunst · maim · xdotool
distro support
arch linux
primary
pacman + yay. everything native. suckless patches preserved.
debian / ubuntu
supported
apt-based. neovim via appimage. suckless compiled from source.
fedora
supported

others
